# 인기 검색어 발굴 전략 가이드

현재 상황: **168,558개**의 키워드를 수집했고, 대부분의 시드 키워드 확장이 완료된 상태입니다.

## 🎯 발굴 전략 우선순위

### 1. **롱테일 키워드 발굴** (가장 효과적) ⭐⭐⭐⭐⭐

**방법:**
- 기존 인기 키워드의 변형/조합 생성
- 예: "일본여행" → "일본여행 추천", "일본여행 가이드", "저렴한 일본여행"

**실행:**
```bash
npx tsx scripts/discover-trending-keywords.ts --strategy=variations
```

**장점:**
- 경쟁이 적고 전환율이 높음
- 검색 의도가 명확함
- 기존 키워드 기반이라 관련성 높음

---

### 2. **카테고리별 인기 키워드 시드 활용** ⭐⭐⭐⭐

**방법:**
- 주요 카테고리별 인기 키워드를 시드로 사용
- 각 카테고리에서 연관 키워드 확장

**실행:**
```bash
npx tsx scripts/discover-trending-keywords.ts --strategy=categories
```

**카테고리 예시:**
- 쇼핑: 패션, 뷰티, 전자제품, 가전, 가구
- 여행/레저: 여행, 호텔, 항공, 렌터카, 관광지
- 건강/의료: 병원, 약국, 운동, 다이어트
- 교육: 학원, 온라인강의, 자격증, 어학
- 금융: 은행, 보험, 투자, 대출
- 부동산: 부동산, 매매, 전세, 월세
- 자동차: 자동차, 중고차, 정비
- 음식/외식: 맛집, 카페, 배달, 요리
- 엔터테인먼트: 영화, 드라마, 게임, 음악
- 스포츠: 축구, 야구, 농구, 골프

---

### 3. **계절성/이벤트 기반 키워드** ⭐⭐⭐⭐

**방법:**
- 계절별, 이벤트별 인기 키워드 수집
- 예: 12월 → 크리스마스, 연말, 선물, 파티

**실행:**
```bash
npx tsx scripts/discover-trending-keywords.ts --strategy=seasonal
```

**계절별 키워드 예시:**
- 봄 (3-5월): 벚꽃, 봄옷, 피크닉, 야외활동
- 여름 (6-8월): 여름옷, 휴가, 수영, 해변, 에어컨
- 가을 (9-11월): 가을옷, 등산, 단풍, 추석
- 겨울 (12-2월): 겨울옷, 스키, 보드, 온천, 크리스마스, 설날

**이벤트 키워드:**
- 블랙프라이데이, 싱글데이, 할인, 세일, 프로모션

## 🚀 실전 실행 가이드

→ 각 카테고리를 시드로 사용하여 확장

### Step 3: 계절성 키워드 수집

- 발견된 키워드 중 검색량이 높은 것부터 수동 수집
- 또는 자동 수집 시스템에 시드로 추가

## 📊 효과적인 키워드 발굴 팁

### 1. **검색 의도 기반 발굴**
- 정보성: "~하는 방법", "~가이드", "~정보"
- 구매성: "~구매", "~가격", "~할인", "~리뷰"
- 비교성: "~비교", "~추천", "~순위"
- 지역성: "~맛집", "~주변", "~근처"

### 2. **경쟁이 적은 롱테일 키워드**
- 3-5단어 조합
- 구체적인 질문 형태
- 예: "서울 강남 맛집 추천", "아이폰 15 프로 가격 비교"

### 3. **최신 트렌드 반영**
- 신제품 출시: "아이폰 16", "갤럭시 S25"
- 인기 드라마/영화: "오징어 게임", "기생충"
- 사회 이슈: "코로나", "원격근무", "재택근무"

### 4. **기존 키워드 분석**
- 검색량은 높지만 문서수가 적은 키워드
- 경쟁이 적은 틈새 키워드
- 관련 키워드가 적은 키워드
